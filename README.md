# Game Recommendation Database System 🎮

This repository contains the database architecture and SQL implementation for a Game Recommendation System, developed as a course project at **KNEU**.

## 🎯 Project Objective
The goal is to design and implement a relational database that stores information about video games, users, and genres to provide personalized recommendations.

## 🛠️ Tech Stack
* **DBMS**: Microsoft SQL Server
* **Tool**: SQL Server Management Studio (SSMS) 22
* **Language**: SQL (T-SQL)
